# ATLAS-DES-005: Avionics Operator UI Design

## 1. Visual Style
- **Background**: Deep Grey/Black (#121212).
- **Text**: Cyber-Green (#00FF00) for active, Amber (#FFA500) for standby.
- **Font**: Robot Monospaced or Courier New for alignment.

## 2. Component Layout
- **Top Row**: System Status & Fault Indicators.
- **Mid Section**: Large Digital Readouts for configured parameters.
- **Bottom Row**: Master Arm (Activate) and Emergency Stop (Cancel).

## 3. Feedback
- No system dialog popups.
- Errors appear as "Advisory Alerts" within the UI frame.
