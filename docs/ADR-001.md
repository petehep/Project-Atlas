# ADR-001: Project Documentation & Engineering Standards

## Context
Project Atlas requires high reliability and traceability (GAMP 5 alignment). 

## Decision
1. Use Markdown (.md) as the Single Source of Truth for documentation in Git.
2. Use professional numbering (URS, FRS, SDS) for traceability.
3. Deliver PDFs for archival/distribution.

## Status
Accepted - 2026-06-30
