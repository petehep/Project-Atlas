## 1. Introduction
This GAMP 5-aligned URS defines the core operational requirements for the Atlas flightline clock.

## 2. Requirements Traceability

### 2.1 Operational (UR-OP)
- **UR-OP-01**: Supports Concurrent State Model (Active vs. Next).
- **UR-OP-02**: Atomic "Activation" mechanism.
- **UR-OP-03**: Prevention of double-activation mid-heat.

### 2.2 Display & Timing (UR-DT)
- **UR-DT-01**: Real-time PC sync (HH:MM:SS).
- **UR-DT-02**: Staged countdown sequence (Insertion -> Heat).
- **UR-DT-03**: Deterministic timing to prevent drift.

### 2.3 Performance & Safety (UR-PS)
- **UR-PS-01**: Master Cancel / Zero-Reset capability.
- **UR-PS-02**: State persistence for recovery.
