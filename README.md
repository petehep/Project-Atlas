# Project Atlas

## Master Flightline Clock

Project Atlas is the internal development codename for the Master Flightline Clock.

The Master Flightline Clock is a mission-critical timing and display system designed to manage competitive gliding race heats for the Southern Soaring League.

The software provides:

- Real-time race timing
- Pilot insertion window management
- Heat countdown management
- Dual-display operator and public interfaces
- Deterministic state-based operation
- Comprehensive event logging
- Expandability for future competition requirements

## Project Objectives

Atlas is being engineered as a professional-grade software product using modern software engineering principles.

Primary goals are:

- Reliability
- Simplicity
- Maintainability
- Expandability
- Deterministic behaviour
